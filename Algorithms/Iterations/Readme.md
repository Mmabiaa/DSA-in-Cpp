## Iterations
An iteration is a process of repeating or looping a set of instructions until a specific condition is met.

## Types of Iterations
1. For Loop
2. While Loop
3. Do-While Loop

## For Loop
A for loop is a type of iteration where the loop is executed a specific number of times.

## While Loop
A while loop is a type of iteration where the loop is executed until a specific condition is met.

## Do-While Loop
A do-while loop is a type of iteration where the loop is executed at least once.

## Categories of Iterations
1. Linear Iterations
2. Binary Iterations
3. Multiple Iterations

## Linear Iterations
Linear Iterations is a type of iteration where the loop is executed a once.

## Binary Iterations
Binary Iterations is a type of iteration where the loop is executed twice.

## Multiple Iterations
Multiple Iterations is a type of iteration where the loop is executed more than twice.

## Applications of Iterations
1. Factorial of a number
2. Fibonacci Series
3. Tower of Hanoi
4. GCD of two numbers

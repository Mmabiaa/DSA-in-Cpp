# Recursive Algorithms
A recursive algorithm is an algorithm that calls itself to solve a problem.

## Applications of Recursive Algorithms
1. Factorial of a number
2. Fibonacci Series
3. Tower of Hanoi
4. GCD of two numbers

## Types of Recursive Algorithms
1. Tail Recursion
2. Head Recursion
3. Tree Recursion
4. Indirect Recursion
5. Direct Recursion

## Tail Recursion
Tail Recursion is a type of recursion where the recursive call is the last operation in the function.

## Head Recursion
Head Recursion is a type of recursion where the recursive call is the first operation in the function.

## Tree Recursion
Tree Recursion is a type of recursion where the recursive call is more than one time in the function.

## Indirect Recursion
Indirect Recursion is a type of recursion where the recursive call itself from a different function.

## Direct Recursion
Direct Recursion is a type of recursion where the recursive call is from the same function.
